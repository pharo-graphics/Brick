# Installation

## All-in-one versions 
[Brick](https://github.com/pharo-graphics/Bloc) + [Bloc](https://github.com/pharo-graphics/Bloc)

The following script install the latest version of Brick + Bloc

```smalltalk
Metacello new
    baseline: 'Brick';
    repository: 'github://pharo-graphics/Brick/src';
    load
```
